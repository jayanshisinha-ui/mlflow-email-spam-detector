# 📧 MLflow Email Spam Detection System

This project implements an Email Spam Detection system using Machine Learning and MLflow for experiment tracking.  
It classifies emails as Spam or Not Spam and compares multiple machine learning models to determine the best-performing model.

---

## 🚀 Project Overview

Email spam detection is an important application of Natural Language Processing (NLP).  
In this project, multiple machine learning models are trained and evaluated using MLflow to track performance metrics and store trained models.

The system uses text preprocessing techniques and classification algorithms to predict whether an email is spam or not.

---

## 🧠 Features

- Email Spam Classification using NLP
- Multiple Model Comparison
- MLflow Experiment Tracking
- Accuracy Logging
- Model Saving and Versioning
- Easy-to-understand dataset
- Reproducible experiments

---

## 🛠️ Technologies Used

- Python
- Pandas
- Scikit-learn
- MLflow
- TF-IDF Vectorization

## ▶️ How to Run the Project

Run the training script:

python spam_mlflow.py

Start MLflow UI:

mlflow ui

Open browser:

http://127.0.0.1:5000

## 📊 Models Used

The following machine learning models are used:

Naive Bayes
